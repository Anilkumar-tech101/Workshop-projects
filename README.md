# Diamond Reveal Card Game

A two-player strategic card game where players compete to win diamond cards by revealing the highest-value card for each round.

---

## Game Concept

Diamond Reveal is a turn-based card game played between **two players**.  
A set of **diamond cards** is placed in the middle and shuffled. One diamond card is revealed at a time.  
Each player uses their cards strategically to win the revealed diamond.

---

## Game Components

- **Diamond Cards**
  - Placed in the center
  - Shuffled at the start of the game
  - Only one diamond card is revealed per round

- **Player Cards**
  - Each player receives **13 shape cards**
  - Each card has a numeric value
  - Cards can be used only once

---

## Game Setup

1. Shuffle all diamond cards and place them face down in the middle.
2. Deal **13 shape cards** to each player.
3. Players keep their cards hidden from each other.

---

## Gameplay Rules

1. Reveal one diamond card from the middle deck.
2. Both players secretly choose one card from their hand.
3. Players reveal their chosen cards at the same time.

---

## Winning a Round

- The player with the **higher-value card** wins the revealed diamond card.
- The winning player keeps the diamond card as part of their score.

---

## Tie Rule

- If both players reveal cards with the **same value**:
  - The diamond card is **shared equally** between both players.
  - Each player receives half the diamond’s value.

---

## Card Usage

- Once a card is played, it is discarded.
- Discarded cards cannot be reused.
- Strategic card selection is essential.

---

## Game End

The game ends when:
- All diamond cards have been revealed, or
- Both players have used all their cards.

---

## Final Winner

- Each collected diamond contributes to the player’s total score.
- The player with the **highest total diamond value** wins.
- If scores are equal, the game ends in a **draw**.

---

## Strategy Notes

- Save high-value cards for high-value diamonds.
- Predict opponent behavior.
- Smart sacrifices can help secure later wins.
